Este repositorio contiene una implementación de distintos métodos de navegación para robots móviles usando ROS 2 y Python.  
El paquete incluye métodos de planeación basados en grid, RRT y campos potenciales virtuales, aplicados principalmente a simulaciones en `turtlesim` y a un Puzzlebot.

Métodos incluidos

Navegación por Grid
RRT
Campos potenciales virtuales
